## How to run Locally

- Install dependencies

  ```bash
    yarn
  ```

  or

  ```bash
   npm install
  ```

- Create .env.local and change env variables as per the instructions in .env.example

  ```bash
    cp .env.example .env.local
  ```

- Start the server

  ```bash
    yarn dev
  ```

  or

  ```bash
    npm run dev
  ```

- Go to localhost:3000 in your browser. Enjoy!
